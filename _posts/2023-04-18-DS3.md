## 스택
한 쪽 끝에서만 item을 삭제하거나 새로운 item을 저장하는 자료구조<br><br><br>
**후입선출(LIFO)** 원칙하에 삽입과 삭제 수행<br><br>

- push : 새 item을 저장하는 연산<br>
- pop : top item을 삭제하는 연산<br><br>

## 배열로 구현한 ArrayStack 클래스
1. ArrayStack 클래스
``` java
public class ArrayStack<E> {
	private E s[];
	private int top;
	public ArrayStack(){
		s = (E[]) new Object[1];
		top = -1;
	}
	public int size() { return top+1; }
	public boolean impty() { return (top == -1); }
	...
}
```

2. 이외 메소드
- peek (stack의 top항목 내용 리턴)
``` java
public E peek() {
	if(isEmpty()){
		throw new EmptyStackException();
	}
	return s[top];
}
```

- push (새 item을 저장하는 연산)
``` java 
public void push(E newItem) {
	if(size() == s.length) {
		resize(2*s.length);
	}
	s[++top] = newItem;
}
``` 

- pop (top item을 삭제하는 연산)
``` java 
public E pop() {
	if(isEmpty()){
		throw new EmptyStackException();
	}
	E item = s[top];
	s[top--] = null;
	if(size()>0 && size()==s.length/4) {
		resize(s.length/2);
	}
	return item;
}
```
